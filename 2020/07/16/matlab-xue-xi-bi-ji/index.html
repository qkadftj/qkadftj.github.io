Matlab 学习随记

常用函数

  1. who 显示所有variables

  2. whos 显示所有variables的详细信息

    1. clear + 变量名 可以清除某个变量的赋值，直接clear, 可以

  3. 在一个指令后加;,可以不显示该步骤的结果

    1. clc 清除窗口所有的指令

Format : 使用时，输入format 加 形式

  1. short

  2. long

  3. shortE (科学计数法，下同)

  4. longE

  5. bank(财会计数，保留两位)

  6. hex(二进制)

  7. rat(分数形式)

特殊变量和常数

  1. ans 计算到的答案

  2. i, j: 复数

  3. inf: ∞

  4. eps：最小的数字（非0）

  5. NaN :not a number

  6. pi: Π

矩阵/向量的操作

  1. 输入矩阵的Row vector:

    a = [1 2 3 4]

  2. 输入矩阵的Column vector:

    b = [1;2;3;4]

  3. 输入Matrix:

    A = [1 21 6;5 17 9; 31 2 7]

  4. 获取Matrix中数字的两种方法：

    方法一：二维坐标索取 如矩阵A中，A(3,2) => 2，为第三行第二个。即A(m,n) => 第m行，第n个

    当用形如A([a1 a2],[b1 b2])的形式进行索引时，得到的为a1行，a2行，的b1列，b2列的Matrix。

    方法二：一维顺序索取，从上到下，从左到右，如A(8)=>9, A([1 3 5]) => [1 31 17], A([1 3;1 3]) = [1 31;1 31] ，即，不用,时，直接按照一维顺序索取，而这个索取可以直接装在一个Matrix里(如后两例)

  5. 改写Matrix特定位置的值（Colon operator）

    Syntax：

    1. 步长默认为1，j:k 从j到k，close interval.
    2. 设置步长，j:i:k 从j到k，步长为i, close interval.

    example:

    ​ B = 1:5 =>[1 2 3 4 5]

    ​ B = 1:2:5 =>[1 3 5]

    ​ B = [1:5; 2:3:15; -2:0.5:0] => [1 2 3 4 5;2 5 8 11 14; -2 -1.5 -1 -0.5 0]

    ​ str = ‘a’ : 2 : ‘z’ =>[a c e ······ ]

    此外colon operator 可以用来索引Matrix中的一整行

    如：

    A(3,:)可获得矩阵第三行的所有元素，A(3,:) = [] 可进行矩阵第三行的清除。当右端元素个数为A中一行的元素个数时，可进行重新赋值

  6. Array Concatenation 矩阵的串联

    如 A = [1 2; 3 4] B = [9 9;9 9]

    则：C = [A,B] => [1 2 9 9; 3 4 9 9] 横向链接

    ​ C = [A;B] => [1 2; 3 4; 9 9; 9 9] 纵向链接

    ​ 同一行的连结：C = [A:B] => [1 2 3 4 9 9 9 9]

  7. Array Manipulation 矩阵操作

    operations on array: + - * / ^ . ’

    A + B : element addation

    A * B : 矩阵乘法，用后面矩阵的对应列乘以前矩阵的一行的对应元素后相加，然后作为新矩阵的元素

    A .* B: element multiplication

    A / B： 相当于A * inverted(B)

    A./B : element division

    矩阵与数字的运算

    A + b : element operation, 每一个元素加b

    A/b 与A./b 效果相同，每个元素除以b

    A^b 即A 的b次方，对A进行b次连乘积

    A’ 转置

  8. special Matrix

    linespace(x1,x2,N) ：linearly spaced vectors

    生成一个从x1到x2的线性数组，均匀间隔，N为元素个数，默认100

    eye(n): n*n的单位矩阵

    zeros(n1,n2): n1*n2的entry全部为0的矩阵

    ones(n1,n2): n1*n2的entry全部为1的矩阵

    diag([n1,n2,n3,······]): 对角线矩阵，其他位置全部为0

    rand(): 若无参数，返回一个0~1之间的随机数，若有参数(a1,a2)则返回一个a1行a2列的随机数矩阵。

  9. some matrix related functions

    1. max(): 返回矩阵每一列的最大数字，若求整个矩阵中最大的数字，可嵌套使用max(max())

    2. min()也同样

    3. sum(): 返回矩阵每一列数字的加总，若求整个矩阵的数字的和可嵌套使用sum(sum())

    4. mean() 也同理，欲求整个矩阵用mean(mean())

    5. sort() 与 sortrows()

    • sort(A) / -sort(-A) 分别对应着 每一列都按照升序/降序排列
    • sortrows(A) ，默认依据第一列的数值按升序移动每一行，如果第一列的数值有相同的，依次往右比较
    • C = sortrows(A,2)，从某一列开始升序排列
    • C = sortrows(A,-2)，从某一列开始降序排列
    1. size（）返回二维数组，第一个数字是行长，第二个数字是列长
    2. length() 返回行长和列长中最大的一个
